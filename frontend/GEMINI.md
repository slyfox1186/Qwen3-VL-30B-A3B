# Last Updated: 2025-12-03

## Purpose
Next.js 16 frontend with React 19 for Qwen3-VL chat UI, SSE streaming, Zustand state management, and Tailwind CSS.

## Key Files
- `package.json` - Dependencies (Next.js, React, Zustand, Tailwind, Radix UI)
- `next.config.ts` - Next.js configuration
- `tailwind.config.ts` - Tailwind/PostCSS config

## Dependencies/Relations
Contains `src/` (app, components, hooks, stores, types). Connects to backend API at localhost:8080.

## Rules
- **CSS**: You MUST create a dedicated CSS file for EVERY TSX script instead of using inline CSS.
