# Last Updated: 2025-12-03

## Purpose
shadcn/ui Radix UI primitive components styled with Tailwind for consistent UI.

## Key Files
- `button.tsx` - Button with variants (default, destructive, outline, ghost, link)
- `input.tsx` - Text input field
- `textarea.tsx` - Multi-line text input
- `card.tsx` - Card container (Card, CardHeader, CardContent, CardFooter)
- `scroll-area.tsx` - Scrollable container
- `avatar.tsx` - Avatar display
- `skeleton.tsx` - Loading skeleton placeholder
- `sheet.tsx` - Slide-out drawer/modal
- `tooltip.tsx` - Tooltip overlay
- `collapsible.tsx` - Collapsible content section

## Dependencies/Relations
Used by `components/chat/`. Styled with Tailwind, built on Radix UI primitives.
